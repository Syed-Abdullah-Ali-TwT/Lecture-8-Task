# Task 2 - Connecting an App (Read-Only)

## App I connected

Google Drive (connected Gmail too while I was at it, but Drive is what I actually used for this task).

## What I did

Connected Google Drive through Claude's connectors and asked it to pull up my most recent files. It came back with my actual recent Drive files - top one was literally the PDF of this exact assignment, "Lecture 8 Task.pdf", with the real viewed/modified dates attached. It also grabbed a second recent file, an unrelated portfolio-website task from an earlier lecture.

## How I know it was real data

The file it pulled up is genuinely the one sitting in my Drive right now, and the "viewed today" timestamp lined up with when I'd actually opened it that day. So this wasn't guessed or made up, it was pulling directly from my actual account.

## Permission I granted

Read-only access to Google Drive. That means it can search and read file content/metadata but can't edit, delete, or upload anything on its own. I'd only need write access if I wanted it to create or update files for me directly, which I don't need right now, read is plenty.
