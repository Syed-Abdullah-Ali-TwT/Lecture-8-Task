# Skills & Connectors - Lecture 8 Project

## About

This repo has all five projects from the Skills & Connectors assignment - built and tested using Claude.

## Summary of the five projects

1. **Journaling Skill** - a Skill that turns casual, spoken-style descriptions of my day into a consistent, formatted journal entry, so I actually stick with journaling instead of it feeling like a chore.
2. **Connect an app (read-only)** - connected Google Drive with read-only access and pulled real files to confirm it works with actual data, not made up content.
3. **Skill + Connector combined** - one plain sentence that pulls live Drive data and formats it with the journaling Skill, no manual copy-pasting.
4. **Portability** - handed the Skill off to a friend to test with zero explanation from me, proving it works outside my own chat.
5. **Skill audit** - reviewed the official PDF skill, explained what it does in plain English, and wrote a safety verdict.

## Tools used

Claude (claude.ai) for all five tasks, Google Drive as the connected app.

## Folder structure

- `task-1-my-skill/` - SKILL.md, README, prompts, screenshot of it triggering automatically
- `task-2-connect-app/` - README on the app connected and permission granted, screenshot of real data pulled
- `task-3-skill-plus-connector/` - README on the combined workflow, prompt used, screenshot of the result
- `task-4-portable-handoff/` - README on the hand-off test, proof from my friend
- `task-5-skill-audit/` - written safety assessment of the PDF skill
